Project Overview

Developed a comprehensive autonomy stack for the Clearpath Heron M300 unmanned surface vessel (USV) as part of marine robotics research at Universiti Sains Malaysia. The project focused on enabling autonomous navigation and obstacle avoidance in challenging maritime environments, including GPS-denied scenarios.

Key Objectives

  • Implement GPS-denied navigation capabilities using sensor fusion
  • Develop real-time obstacle detection and avoidance system
  • Create ROS 2-based mission planning and execution framework
  • Integrate ArduPilot SITL for pre-deployment simulation testing
  • Ensure robust performance in varying sea conditions

Technical Implementation

Navigation System

The USV's navigation system combines multiple sensors including LIDAR, IMU, and cameras to create a robust positioning solution. I implemented Extended Kalman Filter (EKF) sensor fusion to maintain accurate state estimation even when GPS signals are unavailable or unreliable near structures and in harbors.

Obstacle Detection & Avoidance

Developed a multi-layered obstacle detection system utilizing:

  • LIDAR point cloud processing for detecting static and dynamic obstacles
  • Vision-based detection using computer vision algorithms for waterborne objects
  • Integrated costmap generation for Nav2 path planning
  • Dynamic Window Approach (DWA) for reactive obstacle avoidance

ROS 2 Architecture

Built a modular ROS 2 system with the following key nodes:

  • Perception Node: Processes sensor data and publishes obstacle information
  • Localization Node: Maintains vehicle state estimate using sensor fusion
  • Planning Node: Generates optimal paths using Nav2 stack
  • Control Node: Interfaces with ArduPilot for low-level motor control
  • Mission Manager: Coordinates high-level mission execution

Simulation & Testing

Prior to field deployment, extensive testing was conducted using ArduPilot Software-In-The-Loop (SITL) simulation integrated with Gazebo. This allowed validation of:

  • Navigation algorithms in various scenarios
  • Obstacle avoidance behaviors
  • Mission planning logic
  • Failure mode responses

Results & Impact

The autonomous USV successfully demonstrated:

  • Autonomous waypoint navigation in open water
  • GPS-denied navigation near dock structures with <2m positioning error
  • Dynamic obstacle avoidance with safe minimum distance maintenance
  • Reliable mission execution in sea state 2-3 conditions

Challenges & Solutions

Challenge: Sensor Reliability in Marine Environment

Solution: Implemented sensor health monitoring and automatic fallback to degraded modes when sensors fail. Added water spray filtering for LIDAR data.

Challenge: Real-time Performance on Embedded Hardware

Solution: Optimized ROS 2 node processing pipelines and implemented efficient point cloud downsampling. Utilized ROS 2's DDS for low-latency inter-node communication.
