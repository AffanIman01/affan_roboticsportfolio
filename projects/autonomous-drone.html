Autonomous Drone System

Advanced UAV with autonomous navigation and obstacle avoidance capabilities

Duration: 2023 - Present

Role: Research Assistant

Organization: Universiti Sains Malaysia (USM)

Project Overview

Developed and implemented an autonomous drone system capable of navigation, path planning, and real-time obstacle avoidance. The project involved extensive work with flight controllers, simulation environments, and computer vision integration.

Technical Stack

ROS 2 (Humble) ArduPilot PX4 Gazebo Python C++ Mission Planner QGroundControl

Key Features

  • Autonomous waypoint navigation with GPS integration
  • Real-time obstacle detection and avoidance using computer vision
  • Path planning algorithms (A*, Dijkstra) implementation
  • Hardware-in-the-loop (HIL) and software-in-the-loop (SITL) testing
  • Integration with Pixhawk and APM 2.8 flight controllers
  • Custom firmware development and configuration

Hardware Components

  • F450/F550 drone frames
  • Pixhawk flight controller
  • APM 2.8 autopilot system
  • GPS module for positioning
  • Companion computer (Raspberry Pi/Jetson Nano)
  • Camera modules for vision-based navigation

Development Process

Simulation & Testing: Extensive testing was conducted using Gazebo simulator with both Harmonic and Fortress versions. SITL simulations allowed for rapid prototyping and algorithm validation before hardware deployment.

Path Planning: Implemented advanced pathfinding algorithms including A* and Dijkstra for optimal route calculation in complex environments. These algorithms were optimized for real-time performance on embedded systems.

Obstacle Avoidance: Developed computer vision-based obstacle detection system using camera feeds processed through custom algorithms to ensure safe autonomous flight.

Technical Challenges

  • Optimizing path planning algorithms for real-time embedded system performance
  • Integrating multiple sensors for robust environmental perception
  • Tuning PID controllers for stable flight characteristics
  • Ensuring reliable GPS positioning in various environmental conditions
  • Managing power consumption for extended flight times

Results & Achievements

  • Successfully achieved autonomous waypoint navigation with 95%+ accuracy
  • Implemented effective obstacle avoidance with real-time response
  • Reduced development time through comprehensive SITL/HIL testing
  • Created reusable ROS 2 packages for future UAV projects

Future Enhancements

  • Integration of advanced SLAM algorithms for GPS-denied environments
  • Multi-drone coordination and swarm intelligence
  • Deep learning-based object detection and classification
  • Enhanced battery management and solar power integration
