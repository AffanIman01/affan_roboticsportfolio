Project Overview

Designed and built an active gyroscopic stabilization system to reduce roll motion in small marine platforms. The system uses a spinning flywheel controlled by precision servo motors and real-time feedback from an inertial measurement unit (IMU) to counteract vessel rolling caused by waves and sea conditions.

Objectives

  • Reduce roll motion amplitude by at least 50% in typical sea conditions
  • Implement real-time PID control with adaptive tuning
  • Design custom PCB for sensor integration and motor control
  • Create robust mechanical housing for marine environment
  • Achieve power consumption under 50W for extended operation

Technical Design

Gyroscope Mechanism

The core stabilization mechanism consists of a high-speed flywheel (3000 RPM) mounted in a gimbal assembly. When the vessel rolls, the gyroscope precesses, creating torque that opposes the rolling motion. The gimbal is actively controlled via servo motors to maximize stabilization effectiveness.

Control System

Developed a PID control algorithm running on ESP32 microcontroller:

  • Proportional: Provides immediate response proportional to roll angle
  • Integral: Eliminates steady-state error from constant wave patterns
  • Derivative: Predicts and dampens roll rate to prevent overshoot

The control loop operates at 100Hz, reading IMU data and adjusting gimbal position to generate counteracting torque.

Sensor Integration

Utilized MPU-6050 6-axis IMU providing:

  • 3-axis gyroscope for angular velocity measurement
  • 3-axis accelerometer for tilt angle estimation
  • Complementary filter fusion for accurate roll/pitch estimation
  • I2C interface to ESP32 for low-latency communication

PCB Design

Designed custom control board in KiCad featuring:

  • ESP32-WROOM-32 microcontroller module
  • Dual H-bridge motor drivers for gimbal control
  • IMU sensor mounting with vibration isolation
  • Power regulation (12V to 3.3V/5V rails)
  • Emergency cutoff circuit for safety
  • Status LED indicators and serial debug interface

Implementation Challenges

Challenge: Gyroscopic Precession Prediction

Solution: Implemented mathematical model of gyroscopic dynamics to predict precession torque. Used Euler's equations to calculate required gimbal angles for optimal stabilization force generation.

Challenge: Vibration Interference

Solution: Added low-pass filtering to IMU readings (10Hz cutoff) and mechanically isolated sensor from motor vibrations using rubber mounts. Improved signal-to-noise ratio by 15dB.

Challenge: Power Management

Solution: Optimized motor control PWM duty cycles and implemented intelligent sleep modes during calm sea states. Reduced average power consumption from 75W to 42W.

Results

Testing in controlled wave tank conditions (simulating sea state 3) demonstrated:

  • 62% reduction in roll amplitude compared to unstabilized vessel
  • 90ms average response time from roll detection to corrective action
  • Stable operation over 8-hour continuous testing period
  • Effective stabilization in roll frequencies from 0.2 to 2 Hz
