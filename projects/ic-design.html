Project Overview

Developing expertise in integrated circuit design with focus on System-on-Chip (SoC) architecture, RTL development, and digital verification methodologies. This work encompasses the full IC design flow from specification to layout.

Technical Domains

RTL Design Design Verification (DV) Physical Design (PD) Layout Design Verilog/SystemVerilog FPGA Hardware Description Languages

Key Learning Areas

  • RTL (Register Transfer Level) Design: Creating hardware descriptions for digital circuits using HDLs
  • Design Verification: Developing testbenches and verification environments to ensure design correctness
  • Physical Design: Floor planning, placement, routing, and timing closure
  • Layout Design: Physical implementation of integrated circuits at the transistor level
  • SoC Architecture: Understanding system-level integration and bus architectures

Design Flow Expertise

1. Specification & Architecture: Understanding design requirements and creating architectural specifications for IC components.

2. RTL Development: Writing synthesizable RTL code in Verilog/SystemVerilog following industry best practices and design guidelines.

3. Verification: Creating comprehensive verification environments using SystemVerilog and UVM methodologies to validate design functionality.

4. Synthesis & Implementation: Converting RTL to gate-level netlists and implementing on FPGA or ASIC targets.

5. Physical Design: Floor planning, placement, clock tree synthesis, routing, and timing analysis to meet performance targets.

  • Simulation: ModelSim, VCS, Questa

  • FPGA Tools: Vivado, Quartus
  • Physical Design: ICC2, Innovus
